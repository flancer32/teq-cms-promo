ADSM: управление сайтом через агента

Что это и зачем

ADSM (Agent Driven Site Management) позволяет поддерживать сайт через текстовые инструкции. Вы создаёте продукт, а агент заботится о его отражении в HTML.

Преимущества

  • Обновления без кода и админок
  • Сайт всегда соответствует описанию продукта
  • История правок в git
  • LLM синхронизирует переводы

Как работает

  1. Пишете контекст и задачи
  2. Агент создаёт или меняет страницы
  3. Git фиксирует изменения
  4. Сайт публикует актуальные версии

Что входит в продукт

  • Методология ADSM
  • Шаблоны контекста и инструкций
  • Примеры HTML-страниц
  • Руководство по работе с агентом

Для кого

  • Тем, кто устал ждать помощи разработчика
  • Командам, где продукт быстро меняется
  • Разработчикам, работающим через git и LLM
